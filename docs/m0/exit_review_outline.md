# 阶段 M0 评审提纲

## 1. 项目背景与目标（5 min）
- 现有系统简介、业务痛点
- M0 调研范围与目标回顾

## 2. 当前状态总结（10 min）
- 代码结构与运行流程示意
- 主要问题 & 快速收益项

## 3. 目标架构蓝图（15 min）
- 总体架构图、关键组件职责
- 任务编排/配置管理/数据服务的交互
- 前后端分离方案与演示节点

## 4. 领域模型与配置流程（10 min）
- 实体关系图、数据完整性规则
- Excel 导入 → 审核 → 发布 → 回滚流程

## 5. 约束、风险与决策（10 min）
- 已确认约束与工作假设
- 风险清单与缓解计划
- 关键决策确认

## 6. 下一步计划（5 min）
- M1 重点工作：
  - 初始化 Django 项目、接入 DRF、搭建配置模型
  - 搭建 Celery + Redis，封装首个协议 Worker
  - 前端选型与设计稿输出
- 待补充资源与协作事项
